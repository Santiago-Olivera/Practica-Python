Este es un pequeño programa hecho en Python, mediante el cual se crea una interfaz grafica que permite crear una base de datos para usuarios utilizando Sqlite
Cuenta con un menu en la parte superior en donde se se tienen los campos: 
BBDD : Cuenta con la opción conectar y la opción salir, si conectamos se creara una nueva base de datos para usuarios. 
si ya se tiene una base de datos, nos advierte que ya hay una existente. 

Borrar :  se encarga de borrar los campos entry que esten llenos para escribir nueva información 

CRUD: Cuenta con un submenu que permite, las funciones que crear, leer, actualizar y borrar usuarios. 

Ayuda: Parte del menu destinada para licencia y acerca de... en el momento no hacen nada . 

Luego tenemos el frame con los campos de texto para ingresar o visualizar a los usuarios. 

El campo Id, unicamente se requiere cuando se desea leer algun usuario de la base de datos por su ID. 

En la parte inferior se encuentran cuatro botones que también tienen la funcionalidad CRUD.
